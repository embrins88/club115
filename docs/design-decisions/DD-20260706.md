# Design Decisions

This file documents major project decisions for Club 115.

The goal is to preserve not just what was built, but why certain choices were made.

---

## DD-0001 — Club 115 belongs inside the blisOS ecosystem

**Date:** 2026-07-06

### Decision

Club 115 will remain its own project and repository within the larger blisOS ecosystem.

### Reason

blisOS represents my complete personal engineering ecosystem, including my homelab, portfolio website, documentation, GitHub projects, and other technical initiatives.

Club 115 is one component of that ecosystem: a personal infrastructure management platform.

---

## DD-0002 — Use one repository for desktop and web interfaces

**Date:** 2026-07-06

### Decision

The desktop and web versions of Club 115 will live in the same repository.

### Reason

Both interfaces support the same platform and should share documentation, branding, roadmap, and project history.

This also makes future consolidation easier.

---

## DD-0003 — Use a documentation-first project structure

**Date:** 2026-07-06

### Decision

Club 115 will include documentation from the beginning, including roadmap, changelog, installation notes, diagrams, screenshots, and design decisions.

### Reason

The project is part of a professional portfolio and should demonstrate not only coding ability, but also planning, organization, and technical communication.

---

## DD-0004 — Keep Club 115 personal-use focused

**Date:** 2026-07-06

### Decision

Club 115 is being developed for learning and personal homelab use, not public distribution.

### Reason

The purpose of the project is to solve real problems in my own environment while building practical skills in systems administration, networking, automation, and software development.

---

## DD-0005 — Use version milestones to track maturity

**Date:** 2026-07-06

### Decision

Club 115 will use milestone-based version definitions.

### Version Definitions

- `0.1.x` — Project foundation, repository structure, and documentation
- `0.2.x` — Desktop application working
- `0.3.x` — Core infrastructure management features
- `0.4.x` — Web interface begins
- `0.5.x` — Desktop and web starting to converge
- `0.9.x` — Feature complete and polishing
- `1.0.0` — First stable version where Club 115 successfully manages the homelab as originally envisioned

### Reason

Version numbers should reflect meaningful progress rather than random updates.

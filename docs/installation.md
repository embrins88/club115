# Installation

This document explains how to set up and run Club 115.

> **Current Status:** Version 0.1.0-alpha

At this stage, Club 115 is in the project foundation phase. The repository structure and documentation have been established, but the application has not yet been integrated.

---

## Requirements

The following software will be required as development progresses:

- Python 3.13 or later
- Git
- Visual Studio Code (recommended)
- GitHub Desktop (recommended)

---

## Repository Structure

The project is organized into the following major components:

- `desktop/` — Desktop application
- `web/` — Future web interface
- `docs/` — Project documentation
- `branding/` — Logos, icons, and design standards
- `archive/` — Historical project artifacts

---

## Current Installation

No installation steps are required for Version 0.1.0-alpha.

Application setup instructions will be added once the desktop application has been migrated into the repository.

---

## Future Installation

Future versions of Club 115 will include:

- Dependency installation
- Environment configuration
- First-time setup
- Application startup
- Configuration management

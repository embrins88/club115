# Club 115 Roadmap

This roadmap tracks the development of Club 115 as a personal infrastructure management platform within the blisOS ecosystem.

---

## Milestone 1 — Foundation (`v0.1.x`)

### Goal

Create a clean, professional repository structure before adding application source code.

### Tasks

- [x] Create GitHub repository
- [x] Establish project folder structure
- [x] Add README
- [x] Add MIT License
- [x] Add Python `.gitignore`
- [x] Create documentation files
- [x] Start design decision log
- [ ] Complete roadmap
- [ ] Complete changelog
- [ ] Add installation notes
- [ ] Define initial branding/colors

### Definition of Done

The repository is organized, documented, version controlled, and ready to receive the existing Club 115 application source code.

## Milestone 2 — Desktop Application (`v0.2.x`)

### Goal

Move the existing desktop application into the repository and make it runnable from the `desktop` directory.

### Tasks

- [ ] Review existing desktop app files
- [ ] Move source code into `desktop/`
- [ ] Remove obsolete files
- [ ] Organize pages, services, components, and assets
- [ ] Verify the application runs

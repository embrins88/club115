#  🌿Club 115

> A personal infrastructure management platform built to simplify administration, documentation, and automation within my homelab.

**Status:** Active Development  
**Current Version:** 0.1.0-alpha

---

## Overview

Club 115 is my long-term infrastructure project, developed as both a learning platform and a practical tool for managing my self-hosted environment.

Rather than relying on multiple disconnected applications, the goal is to bring infrastructure management, documentation, automation, and monitoring into a single, cohesive platform.

Every feature is built to solve a real problem in my own environment while helping me develop practical skills in systems administration, networking, Python development, and infrastructure design.

---

## Why I Built It

Like many homelab enthusiasts, I found myself juggling multiple tools to administer different parts of my environment.

Club 115 began as a simple dashboard, but has evolved into a unified platform that reflects both the growth of my homelab and my own journey through Information Technology.

Although this project is designed for personal use, it also serves as a living portfolio documenting my approach to designing, building, and maintaining technical systems.

---

## Overview

Club 115 is a long-term project focused on building a centralized paltform for managing a self-hosted homelab.

The goal is to provide a clean, intuitive interface for monitoring infrastructure, accessing services, documenting the environment, and streamlining administrative tasks from a single location.

Rather than being a collection of unrelated utilities, Club 115 is being developed as a unified platform that grows alongside the infrastructure it manages.

---

## Project Goals

- Create a modern desktop application for homelab management.
- Develop a browser-based interface for remote access.
- Centralize infrastructure documentation.
- Simplify routine administration tasks.
- Build a modular platform that can expand over time.
